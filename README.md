# simple_image_viewer

Edit setting
```javascript
// e.g.
// ./画像ファイル名_001.jpg
var file_path         = "./";
var file_name         = "画像ファイル名";
var file_extension    = ".jpg";
var zero_suppress_num = 3;
```

- 画像の右の方をクリックすると次のページ
- Click on the right side of the image => next page
- 画像の左の方をクリックすると前のページ
- Click on the left side of the image => Previous page
